# Face Expression/Age/Gender Recognizer

## How it Works

Your webCam provides input to the tenflow model which are embed in this code. and based on results it provides output.It could be complete written in javascript.

## Features

- Face Recognition
- Face LandMarks
- Face Expressions
- Age and Gender
